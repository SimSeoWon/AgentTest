# 2026-03-28 14:55 — mcp/ 경로를 .claude/ 하위로 이동

## 작업 개요
프로젝트 루트에 노출되던 `mcp/` 디렉토리를 `.claude/mcp/`로 이동하여
에이전트 관련 파일을 `.claude/` 안에 통합.

## 수정 파일
- `build.bat` — `--distpath "dist\.claude\mcp"` 으로 변경
- `watcher/agent_templates.py` — MCP 경로 `./mcp/` → `./.claude/mcp/` (전체 치환)
- `INSTALL.md` — 배포 구조 다이어그램 갱신
- `CLAUDE.md` — 배포 구조 다이어그램 갱신

## 변경 전/후
```
변경 전                       변경 후
[프로젝트 루트]/              [프로젝트 루트]/
├── watch.exe                 ├── watch.exe
├── mcp/           →          └── .claude/
```
