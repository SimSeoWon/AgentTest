## 주요 설계 결정

### 순환 import 방지
- `common.py`에 모든 공용 함수/상수 배치
- 모든 모듈이 `import common`으로 접근
- `_server_mode` 등 전역 변수는 `common._server_mode`로 attribute 접근 (바인딩 복사 방지)

### PyInstaller 호환
- 기존 `--paths "watcher"` 설정으로 충분
- `build.bat` 수정 불필요
- watch.py가 모든 모듈을 직접/간접 import → PyInstaller 자동 추적
