# 2026-03-28 14:30 — 머신 특정 경로 제거

## 작업 개요
다중 사용자 협업을 위해 하드코딩된 절대 경로를 환경변수 기반으로 교체.

## 수정 파일
- `mcp/crash_analyzer/server.py`

## 변경 내용

### 문제
`cdb.exe` 탐색 경로가 `C:\Program Files` 로 하드코딩 → Windows 설치 드라이브가 다른 PC에서 탐색 실패 가능.

### 해결
`_find_cdb()` 탐색 순서를 3단계로 변경:
1. `PATH`에 등록된 `cdb.exe` (`shutil.which`)
2. `%ProgramFiles(x86)%`, `%ProgramFiles%`, `%ProgramW6432%` 환경변수 기반 경로
3. 없으면 `None` 반환 → XML/로그 폴백

모든 소스 파일에서 머신 특정 절대 경로 없음 확인 완료.
