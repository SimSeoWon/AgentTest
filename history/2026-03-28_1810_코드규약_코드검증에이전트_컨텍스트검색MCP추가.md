# 03_코드규약 / 05_코드검증 에이전트에 context-search MCP 추가

## 작업 개요
코드 리뷰 파이프라인에서 사용되는 03_코드규약, 05_코드검증 에이전트에 `context-search` MCP를 등록하여
`combined_search` 통합 검색(벡터+태그)을 활용할 수 있도록 개선.

## 배경
- 기존에 `context-search` MCP는 02, 07, 08, 09, 10 에이전트에만 등록되어 있었음
- 코드 리뷰 핵심 에이전트인 03(규약 검증), 05(버그/안전성 검증)는 관련 컨텍스트를 참조할 수 없었음
- 컨텍스트를 참조하면 기존 코드 패턴과 비교하여 더 정확한 리뷰가 가능

## 변경 전후

| 에이전트 | 변경 전 mcpServers | 변경 후 mcpServers |
|----------|-------------------|-------------------|
| 03_코드규약 | gemini-query | **context-search** + gemini-query |
| 05_코드검증 | gemini-query | **context-search** + gemini-query |
