# 2026-03-28 14:25 — .gitignore 추가

## 작업 개요
빌드 산출물과 Python 캐시 파일을 git에서 제외하는 `.gitignore` 추가.

## 신규 파일
- `.gitignore`

## 제외 항목
| 항목 | 이유 |
|------|------|
| `dist/` | PyInstaller 빌드 결과물 — 직원에게 직접 전달하는 배포 파일 |
| `build/` | PyInstaller 임시 빌드 파일 |
| `*.spec` | PyInstaller 자동 생성 스펙 파일 |
| `__pycache__/`, `*.pyc` | Python 바이트코드 캐시 |
