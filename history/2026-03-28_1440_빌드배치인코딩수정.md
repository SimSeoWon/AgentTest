# 2026-03-28 14:40 — build.bat 인코딩 오류 수정

## 작업 개요
build.bat 실행 시 echo 문의 한글이 cmd에서 명령어로 오인되는 인코딩 오류 수정.

## 수정 파일
- `build.bat`

## 원인
Windows cmd.exe는 배치 파일의 한글 echo 문을 올바르게 처리하지 못하는 경우가 있음.
특히 UTF-8로 저장된 배치 파일에서 한글이 잘려 명령어로 인식됨.

## 해결
- `chcp 65001 >nul` 추가 (UTF-8 코드 페이지 설정)
- 모든 echo 문을 영문으로 교체
