- `mcp/commandlet_runner/server.py` — 신규 MCP 서버 (커맨드렛 실행 전용)

- `watcher/agent_templates.py` — 10_에셋검증 에이전트 추가

## 주요 변경 내용

### 신규: `mcp/commandlet_runner/server.py`
- `find_unreal_editor(project_root)` — .uproject EngineAssociation → winreg → %ProgramFiles% 탐색
- `run_data_validation(project_root, timeout)` — DataValidation 커맨드렛 실행
- `run_commandlet(commandlet, project_root, extra_args, timeout)` — 임의 커맨드렛 실행

### `watch.py` 변경
- `ASSET_EXTENSIONS = {'.uasset', '.umap'}` 상수 추가
- `load_or_init_config()` — `auto_asset_validation` 설정 추가 (최초 설정 시 y/n 입력)
- `_find_uproject()` — base_dir에서 .uproject 탐색
- `_find_unreal_editor()` — winreg + 환경변수 기반 UnrealEditor-Cmd.exe 탐색
- `run_asset_validation()` — DataValidation 커맨드렛 실행 → Claude 분석 → reviews/ 저장
- `main()` — `auto_asset_validation` 설정에 따라 `run_asset_validation()` 조건부 호출

### 에셋 검증 흐름
```
에셋 변경 감지 (.uasset / .umap)
  → _find_unreal_editor()   (winreg → %ProgramFiles% 탐색)
  → UnrealEditor-Cmd.exe -run=DataValidation -log -unattended -nullrhi
  → _call_claude()          (결과 분석)
      → .claude/reviews/YYYY-MM-DD_HHMM_<hash>_assets.md
```
