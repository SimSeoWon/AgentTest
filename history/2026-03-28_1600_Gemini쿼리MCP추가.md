## 주요 설계

### Graceful Degradation
```python
if not shutil.which("gemini"):
    return "Gemini CLI가 설치되어 있지 않습니다."
```
- 집 PC: gemini 있음 → 정상 동작
- 회사 PC: gemini 없음 → 안내 메시지 반환, 크래시 없음

### 제공 툴
- `gemini_analyze(prompt, timeout)` — 대규모 분석 위임
- `gemini_status()` — CLI 설치 및 버전 확인
